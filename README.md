## 📝 Sobre o Projeto

Este repositório é um guia de configuração e uma coleção de scripts úteis para o **Termux**. O objetivo é facilitar a instalação de ferramentas de desenvolvimento (PHP, Python, Node.js), servidores web e utilitários de sistema diretamente no dispositivo móvel.

### 🔥 Funcionalidades e Usos

* **💻 Ambiente Dev:** Configuração de linguagens como PHP (para rodar Laravel no celular!) e Python.
* **🌐 Servidor Web:** Scripts para iniciar servidores Apache, Nginx ou o servidor embutido do PHP.
* **🛡️ Ferramentas de Rede:** Utilitários para testes de conectividade, SSH e monitoramento.
* **🤖 Autenticação & Git:** Configuração de chaves SSH para gerenciar repositórios do GitHub pelo celular.
* **⚡ Atalhos:** Scripts para automação de tarefas repetitivas no sistema Android.
